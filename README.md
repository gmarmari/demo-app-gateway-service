# demo-app-gateway-service
Just a demo project.
* The gateway service of my demo spring application.
* The UI of the demo sping application written with vue.js. 

 # Java
 It requires Java version 17

# Build the App

```
./mvnw clean package
./mvnw spring-boot:run
 ```
 
 ### Or with docker
 ```
 docker-compose up --build

 ```
